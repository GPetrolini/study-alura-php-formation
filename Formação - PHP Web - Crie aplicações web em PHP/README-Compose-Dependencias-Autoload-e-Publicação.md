# Curso: Composer: Dependências, autoload e Publicação

Este curso é um treinamento completo sobre o Composer, a ferramenta padrão para gerenciamento de dependências no ecossistema PHP. O foco foi em como o Composer funciona internamente e em como utilizá-lo de forma eficiente no dia a dia do desenvolvimento.

## O que aprendemos?

O curso nos proporcionou um conhecimento aprofundado do Composer e suas funcionalidades, abordando desde os conceitos básicos até a publicação de pacotes:

* **Fundamentos do Composer:**
    * O que é o Composer e sua importância como gerenciador de dependências.
    * Processo de instalação e o uso do comando `composer`.

* **Configuração e Autoload:**
    * Compreensão do arquivo `composer.json` e como ele define a estrutura do projeto.
    * Configuração de autoloading utilizando `PSRs`, `classmap` e `files` para carregar classes e arquivos automaticamente.
    * Diferenciação entre dependências de produção (`require`) e de desenvolvimento (`require-dev`).

* **Scripts e Eventos:**
    * Criação e configuração de scripts personalizados no `composer.json` para automatizar tarefas.
    * Desenvolvimento de scripts compostos, que executam outros scripts de forma sequencial.
    * Adição de descrições para tornar os scripts mais claros e documentados.
    * Compreensão e utilização dos eventos do Composer para executar ações em momentos específicos (ex: rodar testes automaticamente ao fazer `composer update`).

* **Publicação e Uso de Pacotes:**
    * Processo de disponibilizar um pacote no **Packagist**, o repositório público de pacotes do Composer.
    * Publicação do projeto no GitHub e no Packagist, incluindo a definição de licença e documentação.
    * Consumir um pacote publicado em um novo projeto usando `composer require` e como mantê-lo atualizado.

* **Distribuição de Binários:**
    * Aprendizado sobre como incluir e configurar um arquivo binário na pasta `bin` do projeto.
    * Permitir que o pacote seja executado diretamente do terminal com um comando simples, tornando a ferramenta mais acessível.

## Conclusão:

Os conhecimentos adquiridos neste treinamento são suficientes para gerenciar de forma profissional as dependências de qualquer aplicação PHP. Este curso fornece uma base sólida para continuar explorando o universo de pacotes e ferramentas disponíveis para a linguagem.
