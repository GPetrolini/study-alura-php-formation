# Estudos Alura - Formação PHP

Este repositório é dedicado ao acompanhamento e armazenamento dos materiais, exercícios e projetos desenvolvidos durante a **Formação PHP da Alura**.

**Objetivo:**

Registrar o progresso e o aprendizado obtido ao longo dos diversos cursos que compõem a jornada de formação em PHP oferecida pela plataforma Alura.

**Estrutura (em construção):**

A organização interna seguirá a sequência das formações e cursos, conforme o roadmap de estudos sugerido pela Alura, para facilitar a navegação e o acompanhamento do conteúdo.
