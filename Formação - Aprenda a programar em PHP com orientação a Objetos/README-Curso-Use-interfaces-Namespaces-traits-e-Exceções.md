# Curso: Use interfaces Namespaces traits e Exceções

Este curso, que encerra a primeira formação em PHP, mergulhou em detalhes avançados da Orientação a Objetos e apresentou práticas robustas para organização do código e tratamento de exceções.

## O que aprendemos?

Evoluímos a nossa aplicação para um modelo mais maduro e organizado, explorando os seguintes conceitos:

* **Abstração e Herança:**
    * Criação de classes e métodos abstratos (`abstract`) para definir contratos e comportamentos obrigatórios para classes filhas.
* **Interfaces:**
    * Definição de interfaces (`interface Avaliavel`) para criar contratos de código, garantindo que diferentes classes implementem métodos específicos de forma consistente.
* **Reutilização de Código com Traits:**
    * Utilização de `traits` (`trait ComAvaliacao`) para reutilizar blocos de código em classes que não compartilham a mesma hierarquia de herança.
* **Organização e Gerenciamento de Código:**
    * Introdução a **Namespaces** para organizar classes de forma lógica e evitar conflitos de nomes.
    * Uso de **Autoload** para carregar classes automaticamente, eliminando a necessidade de múltiplos `require`.
* **Tratamento de Erros e Exceções:**
    * Práticas para lidar com erros comuns, como o `DivisionByZeroError`.
    * Criação e lançamento de **Exceptions personalizadas** (`NotaInvalidaException`) para gerenciar caminhos alternativos e tratar erros específicos da aplicação de forma elegante.

## Conclusão da Formação:

Ao final deste curso, completamos a **Formação: Aprenda a programar em PHP com orientação a objetos**. Adquirimos uma base sólida na linguagem, desde os fundamentos até a aplicação de conceitos avançados de POO, que são indispensáveis para a carreira de desenvolvedor.
