# Curso: Organização e Manipulação de Dados em PHP

Este curso aprofundou os conhecimentos em PHP, focando na organização do código, manipulação avançada de dados e interação com fontes externas, além de uma introdução ao desenvolvimento web.

## O que aprendemos?

Exploramos conceitos essenciais para escrever código PHP mais estruturado, limpo e eficiente:

* **Funções e Sub-rotinas:**
    * Compreensão da diferença entre funções (com retorno) e sub-rotinas (sem retorno).
    * Discussão sobre tipos de dados e o uso de parâmetros (incluindo opcionais) em funções.
* **Manipulação de Dados com Funções Nativas:**
    * Utilização de funções PHP para **manipular arrays**, incluindo ordenação e busca por elementos específicos (ex: menor item).
    * Exploração de funções para **manipular textos e strings**.
* **Organização e Modularização do Código:**
    * Princípios de separação do código entre a **definição de funções** e sua **execução**.
    * Início da discussão sobre organização em pastas para melhor estrutura de projetos.
* **Manipulação de Arquivos e Dados Externos (JSON):**
    * Trabalho com o formato **JSON** (JavaScript Object Notation), incluindo `encode` (codificação) e `decode` (decodificação) para intercâmbio de dados.
    * Habilidades para **ler e escrever em arquivos**, permitindo a manipulação de dados de fontes externas.
* **Tratamento de Erros:**
    * Compreensão dos diferentes **tipos de erros no PHP** e como eles podem auxiliar no processo de desenvolvimento e depuração.
* **Spoiler de Desenvolvimento Web:**
    * Introdução ao funcionamento do **PHP na web**, demonstrando a facilidade e a simplicidade de criar interações web básicas com a linguagem.

## Conclusão:

Este curso solidificou a base para a escrita de código PHP mais modular e robusto, preparando o terreno para tópicos mais avançados, especialmente em Orientação a Objetos e desenvolvimento de aplicações web.
