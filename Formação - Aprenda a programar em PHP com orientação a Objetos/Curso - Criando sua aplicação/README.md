# Curso: Aprendendo a programar em PHP (Básico)

Este documento sumariza os principais tópicos abordados no primeiro curso da formação PHP da Alura, focado nos fundamentos da linguagem.

## O que aprendemos?

Neste curso introdutório de PHP, familiarizamo-nos com a sintaxe básica da linguagem e aplicamos conhecimentos de programação geral. Exploramos os seguintes conceitos:

* **Primeiro Programa:** Começamos com o tradicional "Hello World!", entendendo o uso do `echo` e a exibição de conteúdo fora das tags PHP.
* **Fundamentos:** Aprendemos sobre a declaração e utilização de **variáveis** e seus **tipos de dados**, incluindo a peculiaridade da **string numérica** para cálculos.
* **Desenvolvimento de Programa:** Criamos a ferramenta `screen-match.php`, um simulador de streaming de filmes, onde trabalhamos com:
    * Informações como nome do filme e ano de lançamento.
    * Recuperação de notas de usuário via linha de comando.
* **Estruturas de Controle:**
    * **Repetição de dados (loops):** Utilização de estruturas para processar dados repetidamente.
    * **Estruturas de Decisão:** Implementação de `if` e `match` para controle de fluxo.
* **Arrays:** Trabalhamos com arrays simples e exploramos os **arrays associativos** para organizar dados de forma mais complexa.
* **Funções:** Utilizamos algumas funções embutidas para auxiliar no desenvolvimento.
* **Interação com Usuário:** Aprendemos a **ler dados do teclado** do usuário, fundamental para aplicações interativas.

## Projeto Final / Desafio:

* Implementação do arquivo `conta.php`, que simula uma conta bancária, aplicando todos os conceitos aprendidos (loops, estruturas de decisão, variáveis e diferentes tipos de dados).

Este curso proporcionou um sólido primeiro contato com a linguagem de programação PHP, preparando o terreno para tópicos mais avançados.
